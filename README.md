Bashbullet  
===================================  
 a simple pushbullet notification daemon for linux users written in bash  

### Prerequisite  
 node.js and wscat  
 perl (json_pp)  
 notify-osd  

### Installation  
 Excute this script directly.  
 There is no need to install.  

### Configuration  
 place your api key in ~/.bashbullet/config  
 e.g.  
 API_KEY="my_pushbullet_api_key"
